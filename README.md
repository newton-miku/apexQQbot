# Apex助手QQ机器人

## 程序说明
1. 通过API查询相关数据，并使用qq官方机器人SDK开发完成
1. 程序使用webhook模式接收信息回调（回调地址为`http://0.0.0.0:9000/qqbot`，填入官方后台前推荐自行配置反代https）

## 功能说明

- `/a查询` 查询当前账号绑定Apex账户信息

- `/a查询 <EAID>` 查询EAID的账户信息

- `/a地图` 获取当前地图轮换

- `/a绑定 <EAID>` 绑定EAID
